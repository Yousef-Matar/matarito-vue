## Module import

`MButtonSet` component is dependent on `MButton`.

::: code-group

```vue [Composition API]
<script setup>
import { MButtonSet, MButton } from "matarito-vue";
</script>
```

```vue [Options API]
<script>
import { MButtonSet, MButton } from "matarito-vue";
export default {
	components: { MButtonSet, MButton }
};
</script>
```

:::
