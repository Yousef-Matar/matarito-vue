## Module import

::: info

`MBadge` can also be imported as a directive in-order to connect it to an existing element. See [Badge directive](../../../directives/Badge/features) for details.

:::

::: code-group

```vue [Composition API]
<script setup>
import { MBadge } from "matarito-vue";
</script>
```

```vue [Options API]
<script>
import { MBadge } from "matarito-vue";
export default {
	components: { MBadge }
};
</script>
```

:::
