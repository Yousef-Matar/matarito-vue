## Button Set

Multiple buttons are grouped when wrapped inside an element with `m-button-set` class.

<DemoContainer>
   <div class="m-button-set">
		<MButton label="Save" />
		<MButton label="Delete" />
      <MButton label="Cancel" />
   </div>
</DemoContainer>

```vue
<template>
	<div class="m-button-set">
		<MButton label="Save" />
		<MButton label="Delete" />
		<MButton label="Cancel" />
	</div>
</template>
```
