## Basic

Text to display on a button is defined with the `label` property or the `predefined default slot`.

<DemoContainer>
		<MButton label="Submit" />
		<MButton>Submit </MButton>
</DemoContainer>

```vue
<template>
	<MButton label="Submit" />
	<MButton>Submit</MButton>
</template>
```
